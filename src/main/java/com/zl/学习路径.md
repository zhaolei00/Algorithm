 *          可以和面试官聊，把算法调度实现清楚，fn、gn、sn等多个功能按照黑盒来调用，但是需要描述好每个函数的入参和返回值。
 *
 *      解题技巧一: 先按照简单例子寻找解题思路，再进行抽象化、考虑边界。
